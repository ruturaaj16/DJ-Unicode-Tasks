First let me introduce you to this project.
I have accessed the foreign currency exchange API so that the user can choose a base currency and then see the par values of other countries' currencies with the base currency chosen. The API "Exchange rates API" is a free service for current and historical foreign exchange rates published by the European Central Bank.
